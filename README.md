La pagina Esperienza utilizza `/api/daphne`. Configurare su Vercel:
- `OPENAI_API_KEY`
- `OPENAI_MODEL` = `gpt-5-mini`

Caricare tutti i file nella root del repository e fare un nuovo deploy.
