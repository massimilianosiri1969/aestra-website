# AESTRA Website V4 — Rebuilt Hero

Questa versione ricostruisce completamente la homepage iniziale.

## Nuova impostazione
- il software è il protagonista visivo;
- Daphne è inserita dentro il cockpit operativo, non sovrapposta come immagine decorativa;
- nessuna card fluttuante sul volto;
- nessun testo tagliato o elemento che esce dal layout;
- agenda, CRM, Beauty Journey, marketing, magazzino e BI sono percepibili già nella hero;
- messaggio iniziale di Daphne animato;
- dashboard grande e leggibile;
- versione mobile riprogettata.
